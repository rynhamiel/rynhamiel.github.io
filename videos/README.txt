============================================
KAGURA VTUBER - VIDEO FOLDER
============================================

Place your video files in this folder.

SUPPORTED FORMATS:
- MP4 (recommended)
- WebM
- OGG

HOW TO ADD VIDEOS:
1. Copy your video files to this folder
2. Update the video cards in index.html with:
   - data-video="videos/your-video.mp4"
   - data-title="Your Video Title"
   - Thumbnail image path

VIDEO FILES REFERENCED IN WEBSITE:
- stream1.mp4 - The Summoning Ritual (Debut Stream)
- stream2.mp4 - Exorcising Demons in Dark Souls
- stream3.mp4 - Drawing Your OCs LIVE
- stream4.mp4 - Reacting to Creepy Pasta at 3AM

TIPS:
- Use MP4 format for best compatibility
- Keep file sizes reasonable (under 100MB recommended)
- Use descriptive filenames
- Create thumbnails in the images/ folder
